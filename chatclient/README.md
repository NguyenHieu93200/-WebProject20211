# chatclient

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run test
```

### Compiles and minifies for production
```
npm run build
```
After run build, project will be built to dist folder. You can serve it by using serve command:
````
npm install -g server
serve -s dist -l 8080
````
